# 🧠 Skills Demonstrated — Full Stack E-Commerce Project

This document highlights the technical skills implemented in the development of this full-stack e-commerce application.

---

## 🎯 Frontend Development

### Core Technologies
- React (Vite)
- JavaScript (ES6+)
- HTML5, CSS3
- Tailwind CSS

### Skills Applied
- Component-based architecture
- State management using Context API
- Client-side routing using React Router DOM
- API integration using Axios
- Responsive UI design (mobile-first approach)
- Dynamic rendering of product data
- Form handling (authentication and checkout forms)

---

## ⚙️ Backend Development

### Core Technologies
- Node.js
- Express.js

### Skills Applied
- RESTful API development
- Middleware creation and usage
- JWT-based authentication system
- Password hashing using bcrypt
- Role-based access control (User / Admin)
- Structured error handling
- Secure backend architecture

---

## 🗄️ Database Management

### Technology
- MongoDB (Mongoose ODM)

### Skills Applied
- Schema design for real-world applications
- CRUD operations (Create, Read, Update, Delete)
- Data modeling for Users, Products, and Orders
- Relationship handling between collections
